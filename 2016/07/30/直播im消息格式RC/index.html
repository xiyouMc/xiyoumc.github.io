直播IM消息格式

数据结构主要包括：
userId、userName、userIcon、level、msgType、msgContent等

  • userId ：用户ID，唯一标示
  • userName: 用户名
  • userIcon：用户头像
  • level：用户等级
  • msgType：消息类型，包括文字、礼物、点赞、弹幕
  • msgContent：消息内容

主要的消息类型有:

msgType 描述
0 系统消息
1 用户进入直播间
2 文字消息
3 关注主播
4 弹幕消息
5 送礼物
  • 以下是具体的消息描述：

1、系统消息

系统消息是指由服务器或者系统默认发送的消息。

  • 示例

    {
       msgType:0,
       msgContent:系统提醒：禁止黄赌毒

 }
  • 参数
名称 类型 描述 可选 默认值
msgType int 消息类型 N null
msgContent string 消息内容 N null

2、用户进入直播间

用户点开直播间后，聊天室显示该用户的进入消息。

  • 示例

    {
 userId:001,
    userName:mc,
 userIcon:xxxx.jpg,
 level:100,
 msgType:1,
 msgContent:系统提醒：用户mc进入直播间
}
  • 参数
名称 类型 描述 可选 默认值
userId string 用户ID N null
userName string 用户名 N null
userIcon string 头像地址 Y null
level string 用户等级 N null
msgType int 消息类型 N null
msgContent string 系统提示消息信息 N null

3、用户发送文字

用户自己在聊天室内发送文字

  • 示例

    {
    userId:001,
    userName:mc,
    level:100,
    msgType:2,
    msgContent:消息内容
   }
  • 参数
名称 类型 描述 可选 默认值
userId string 用户ID N null
userName string 用户名 N null
level string 用户等级 N null
msgType int 消息类型 N null
msgContent string 系统提示消息信息 N null

4、关注主播

用户关注主播之后，在主播间看到的信息

  • 示例

    {
    userId:001,
    userName:mc,
    level:100,
    msgType:3,
    msgContent:消息内容
   }
  • 参数
名称 类型 描述 可选 默认值
userId string 用户ID N null
userName string 用户名 N null
level string 用户等级 N null
msgType int 消息类型 N null
msgContent string 提示的关注信息 N null

5、用户发送弹幕

发送弹幕功能

  • 示例

    {
    userId:001,
    userName:mc,
    level:100,
    msgType:4,
    msgContent:消息内容
   }
  • 参数
名称 类型 描述 可选 默认值
userId string 用户ID N null
userName string 用户名 N null
level string 用户等级 N null
msgType int 消息类型 N null
msgContent string 弹幕内容 N null

6、送礼物

发送弹幕功能

  • 示例

    {
    userId:001,
    userName:mc,
    level:100,
    msgType:5,
    giftId:001,
    giftCount:100
    msgContent:消息内容
   }
  • 参数
名称 类型 描述 可选 默认值
userId string 用户ID N null
userName string 用户名 N null
level string 用户等级 N null
msgType int 消息类型 N null
msgContent string 弹幕内容 N null
giftId string 礼物id N null
giftCount string 同一个礼物的个数 (x 100) Y null
